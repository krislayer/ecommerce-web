## 📝 Recarga el navegador

Si ya tienes `npm run dev` corriendo:
1. Refresca la página en el navegador
2. Ve a `/catalogo` para ver los productos
3. Haz clic en cualquier producto para ver detalles
4. Agrega productos al carrito

## 🛒 Probar el carrito

1. Ve a `/catalogo`
2. Haz clic en cualquier producto
3. Selecciona talla y cantidad
4. Agrega al carrito
5. Ve al checkout

## 💡 Próximos pasos

Para conectar con Firebase real:
1. Configura `.env.local` con tus credenciales
2. Ejecuta `npm run seed` para poblar Firestore
3. Reemplaza `sampleProducts` con datos de Firestore
